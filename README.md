### Introduction ###
Ubuntu Cleaner is a tool that makes it easy to clean your Ubuntu system. Ubuntu Cleaner can free up disk space and remove private information from your computer.

- Clear browser cache
- Clear APT cache
- Clear thumbnail cache
- Remove unneeded packages
- Remove old kernels
- Remove old installers

### Installation ###
Install from source:
```
$ git clone https://github.com/nuclearcat/ubuntu-cleaner.git
$ cd ubuntu-cleaner
```
### Development ###
$ # To run the program from source:
```
$ ./ubuntu-cleaner
```
To run the unit tests:
```
$ make test
```
To create a .deb package:
```
$ make deb
$ make clean
```
